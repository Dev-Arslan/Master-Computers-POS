Master Computers POS

A lightweight Point of Sale (POS) and inventory management application built with C# (.NET Windows Forms) and MySQL. The application is designed for small computer shops to manage inventory, sales, vendor records, investments, and daily financial summaries through a simple desktop interface.

Features
💻 Product inventory management
🏷️ Vendor management
💰 Sales recording
📦 Stock tracking
🔢 Serial number management
📅 Purchase and sale date tracking
📈 Profit calculation
💵 Cash & bank balance management
📊 Opening and closing balance tracking
📉 Investment tracking
🖨️ Print functionality
📷 Barcode/Scanner support
🗄️ MySQL database integration
Tech Stack
Language: C#
Framework: .NET Windows Forms
Database: MySQL
IDE: Visual Studio
Purpose

This project was developed as a desktop application for managing the daily operations of a small computer retail business. It simplifies inventory management, sales tracking, financial calculations, and vendor management through an easy-to-use Windows interface.
